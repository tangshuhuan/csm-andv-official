二次确认框 ​

基础用法 ​

二次确认框基础实用，通过type值修改图标，目前支持一下四种，若不是一下四种，自行通过插槽传入

<template>
  <div>
    <CsmButton @click="cofirmHandle('info')" type="default">信息</CsmButton>
    <CsmButton @click="cofirmHandle('error')" type="default" style="margin: 0px 10px;">错误</CsmButton>
    <CsmButton @click="cofirmHandle('risk')" type="default"  style="margin-right: 10px;">风险</CsmButton>
    <CsmButton @click="cofirmHandle('highRisk')" type="default">高危</CsmButton>

    <CsmConfirm v-model:open="confirmVis" :title="'提示'" :type="confirmType" @closeModel="modelHandleProcess" >
      <template #content>qqqqq</template>
        <template #footer>
          <a-button @click="modelHandleProcess">取消</a-button>
          <a-button type="primary" >确定</a-button>
      </template>
    </CsmConfirm>
  </div>
  
  <!-- 二次确认框 -->
</template>
<script setup>

import {ref} from 'vue' 
    let confirmVis = ref(false)
    let confirmType = ref('info')
// 二次确认框
const cofirmHandle = (type) =>{
  console.log(type, confirmVis.value);
  confirmVis.value = true
  confirmType.value = type
}


const modelHandleProcess = () =>{
  confirmVis.value = false
}
</script>
<style scoped>
.ant-modal {
  width: 400px;
}
</style>

2、配置参数 ​

参数说明类型默认值
title标题string''
type图标展示类型stringinfo、risk、error、highRisk
content内容插槽slot-
footer底部插槽slot-
titleIcon自定义图标插槽slot-
otherHandle协议类，多选款插槽slot-
promptContent警告内容插槽（红色字体）slot-
otherSelect弹窗类其他操作、例如下拉框slot-
