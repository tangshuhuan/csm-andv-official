快速开始 ​

  • 初始化框架
npm create ***
  • 安装 pnpm
npm install pnpm -g
  • 安装依赖
pnpm install csm-andv
  • 打包示例组件库

注意: 这里需要先执行一次打包,因为组件库文档引用了打包后的文件

pnpm run ***:build
  • 启动测试项目
pnpm run ***:dev
